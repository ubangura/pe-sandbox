# pe-sandbox

Hands-on experiments and code-alongs from my production engineering self-study.

## Structure

| Phase               | Topics                                                             |
| ------------------- | ------------------------------------------------------------------ |
| `01-foundations/`   | Docker, Nginx, HTTP, Gunicorn                                      |
| `02-reliability/`   | Health checks, graceful degradation, validation, postmortems       |
| `03-observability/` | JSON logging, Prometheus, Grafana, Alertmanager                    |
| `04-scalability/`   | Redis caching, HyperLogLog, consistent hashing, Streams, DB tuning |
| `05-cicd/`          | GitHub Actions, deploy scripts, security scanning                  |
| `06-advanced/`      | Chaos testing, capacity planning, incident response                |
